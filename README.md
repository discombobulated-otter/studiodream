
# Studio Portfolio

Premium photography studio portfolio built using **React**, **Tailwind CSS**, and **Framer Motion**. Designed for modern UI/UX, motion-rich interactions, responsive gallery presentation, and optimized performance.

---

## Features

* Motion‑driven UI (page transitions, scroll animations, micro-interactions)
* Responsive gallery with lazy loading and optimized image pipeline
* Modular, maintainable component architecture
* Performance‑optimized build (code splitting, prefetching, asset compression)
* Production‑ready deployment configuration

---

## Tech Stack

**Frontend:** React, Tailwind CSS, Framer Motion
**Build/Deploy:** Vite or Next.js, CDN hosting (Vercel / Netlify / Cloudflare)
**Tooling:** Image optimization workflow, ESLint, Prettier

---

## Project Structure

```
src/
  components/
  pages/ or routes/
  animations/
  assets/
  styles/
  utils/
```

---

## Performance

* Reduced bundle size via code splitting
* Improved LCP/CLS through layout stabilization
* Optimized rendering for image‑heavy components
